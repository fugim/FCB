API Key · football-data.org
Aquesta API admet crides directes des del navegador (sense bloqueig CORS ni de IP). El pla gratuït inclou LaLiga, Champions League i Copa del Rey. No cal targeta de crèdit.
1
Ves a football-data.org/client/register i registra't amb el teu email
2
Rebràs un email amb el teu Auth Token (una cadena de 32 caràcters)
3
Enganxa el token aquí i prem Desar
Per què football-data.org?
A diferència d'API-Sports, football-data.org no bloqueja IPs de proveïdors mòbils ni domèstics. Les crides es fan directament des del navegador sense necessitat de cap proxy.
